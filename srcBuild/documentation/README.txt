Programme de test de mouvement de tondeuses.

Dans le répertoire IN, placer le fichier de commandes et remplacer son nom dans le fichier start.cmd.
Le fichier de résultat est enregistré dans le dossier out.

Les logs du programme se trouvent dans le dossier log.


NB : les fichiers d'entrée et de sortie peuvent se situer ailleurs sur le disque.
